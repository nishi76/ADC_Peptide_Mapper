## Background Databases *(one-time setup, ~10 min)*

The app requires pre-built proteome background databases for uniqueness checking.
Run this script once after installation ГўВЂВ” it downloads from UniProt and caches locally:

```r
source("build_background_db.R")
```

This generates three files in the `data/` folder:

| File | Species | Proteins |
|------|---------|----------|
| `bg_human.rds` | Human (*Homo sapiens*, UniProt Swiss-Prot reviewed) | ~20,442 |
| `bg_monkey.rds` | Cynomolgus Monkey | ~1,177 |
| `bg_rat.rds` | Rat (*Rattus norvegicus*) | ~8,231 |

## Features

- **FASTA input** ГўВЂВ” file upload or paste; multi-chain ADC (HC + LC) auto-detected
- **Trypsin digest** ГўВЂВ” KP/RP rule enforced, 0 missed cleavages, 6ГўВЂВ“30 AA peptides
- **Fixed mod** ГўВЂВ” Carbamidomethylation (CAM) on all C (+57.021 Da)
- **Variable mods** ГўВЂВ” Oxidation (M), Propionamide (C), NEM (C), Drug-linker payloads (MMAE / DM1 / DXd / SN-38 / custom Da)
- **Special mods** ГўВЂВ” Deamidation, Pyroglutamate, Acetylation, Phosphorylation, custom builder
- **Uniqueness check** ГўВЂВ” flags peptides absent from Human / Cynomolgus Monkey / Rat backgrounds at MC = 0, 1, or 2
- **Transition list** ГўВЂВ” MRM (2+/3+, b/y, top 5) and HR (2+/3+/4+, b/y/a, top 6) modes with auto-calculated collision energies
- **Export** ГўВЂВ” Skyline CSV (all peptides or unique only) + Excel summary workbook
